 # Better Call SpaceLLM 🛸⚖️

> AI agent for international space law — multilingual RAG pipeline

Better Call SpaceLLM is an AI-powered agent that answers questions about 
international space law, based on a curated corpus of UN treaties, 
national space laws, and multilateral agreements (EN/FR/RU/ZH/ES).

## Tech Stack
- **AI/ML** : LangChain · Llama · ChromaDB · HuggingFace
- **Backend** : FastAPI · Python
- **Frontend** : React/Streamlit · HTML · CSS (TailwindCSS)
- **Data** : UNOOSA · NASA · ESA · ITU · ASTRO (.pdf, .txt , .docx , etc.)
